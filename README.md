# 📸 Photo Curator - 사진 선별 도구

대량의 사진(1500장+)에서 원하는 수(예: 33장)만큼 라운드별로 선별하는 PyQt5 데스크톱 앱입니다.

## 설치 & 실행

```bash
pip install PyQt5
python photo_curator.py
```

## 사용 흐름

1. **네이버 mybox에서 사진 다운로드** → 한 폴더에 모아두기
2. **앱 실행** → 폴더 선택 → 목표 수 입력 (기본 33장) → 시작
3. **사진 1장씩 보면서 Yes/No** → 라운드 끝나면 선택된 것만 다음 라운드
4. **목표 수 이하가 될 때까지 반복** (예: 1500 → 500 → 150 → 33)
5. **최종 선택 사진을 원하는 폴더로 복사**

## 단축키

| 키 | 동작 |
|---|---|
| **Space** | 선택 |
| **→** | 패스 |
| **←** | 이전 사진 |
| **U** | 선택 토글 |

## 주요 기능

- **썸네일 캐싱**: 15MB+ 고해상도 사진도 빠르게 로드
- **세션 자동 저장**: 중간에 닫아도 이어서 가능
- **다크 테마**: 사진 감상 최적화 UI
- 원본 사진은 절대 수정되지 않음 (읽기 전용)
